CSS Shadows and Outlines

I am a box

Shadow Syntax:-


h-offset v-offset blur spread color inset;

Here;
  • h-offset and v-offset: Determines the shadow's horizontal and vertical position
  • blur: The larger the value, the blurrier the shadow
  • spread: Exparids or shrinks the shadow size
  • color: Defines the shadow color
  • insert: Makes the shadow inner
Shadows can be applied to both boxes and texts- This is an example of text shadow.

Outlines Syntax:-

outline : width style color;

  • width- Sets outline width
  • style- Determines the style of the outline such as solid,dottedor dashed.
  • color-Sets the outline color

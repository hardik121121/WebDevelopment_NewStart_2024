Css Fonts, Text and Color Properties

Fonts :

  • font family is to insert both primary and secondary font options to the webpage.
  • font style is used to italicise the font.
  • font weight is used to make the font strong and emphasize on its text.
  • text decoration is used for underlyining the text.
  • Font size is used to set the size of font according to user needs.
  • line height is used to increase space in linning.
  • letter spacing is used to provide space between letters

    • and many more......

    Text Properties :

  • text transform is used to capitalize, lowercase or uppercase the letters in the provided text.
  • text decoration color is used to change the color of line provided by text decoration.
  • text decoration style can change the type of line style.
  • text decoration thickness can change the thickness of line.
  • text indent is used to provide indendation in the text.
  • text overflow is useful while box creations

    • and many more.......

Colors :

    Colors can be represented in 3 ways :

  • Hex Color Code
  • RGB
  • Color Keywords
  • RGBA
  • HSL

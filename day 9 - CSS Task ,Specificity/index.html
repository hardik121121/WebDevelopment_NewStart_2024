Specificity in CSS

To remove conflict issues Css uses "The Cascade Algorithm"; This Algorithm determines that which property will be applied in css and which property won't.


Since there are multiple color properties applied in the same element so which property will be winning?

    There are 4 rules according to the cascade algorithm :

  1. Position and order of appearance: the order in which your css rules operator.
  2. Specificity: an algorithm that determines which css selector has the strongest match.
  3. Origin: the orderin which css appears and where it comes from, wether that is a browser style, css from a browser extension, on your authored css
  4. Importance: Some css rules are weighted more heavily than others, especially with the important rule type.

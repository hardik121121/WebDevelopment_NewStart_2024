CSS Box Model

Four main parts of css box model are :
  1. content
  2. padding
  3. border
  4. margin
  1. After giving padding the text is shifted by 10px from all sides
  2. Margins are used to create space around elements, outside of any defined borders.
  3. Border adds a border of desired length.
  4. Box-sizing border box assigns the total height or width - is the total of content + border + padding
  5. The concept of margin collspse says that if margin of two elements are collapsing the margin of greatest element will be applied once.

To calculate the total width and height of an element :

  • Total Width: Width + left border + right border + left margin + right margin + left padding + right padding
  • Total Height: Height + bottom border + top border + top padding + bottom padding + top margin + bottom margin

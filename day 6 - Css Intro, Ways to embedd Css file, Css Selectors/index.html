Introduction to CSS(Cascading Style Sheets)


Html is used to define skeleton of a webpage whereas, CSS can be embedded in a html file and can also be made an external file for styling the websites.

How can a css file be embedded in html?

There are 3 ways to embedd a css into html file:

  1. Inline Css
  2. External Css
  3. Internal Css
  • Inline Css refers to writing style attribute in every individual element as per requirement but is not considered a good practice because individually a programmer will have to alter changes in different sections of the html file in order to get the work done.

  • Internal Css refers to writing style tag in the head section of html file and performing changes in the selector and property-value pair of the tags as per requirement.

  • External Css refers to creating an extternal css file and linking it to the hmtl file eith the help of link tag by providing relation as stylesheet and hyper-reference as the css file name.

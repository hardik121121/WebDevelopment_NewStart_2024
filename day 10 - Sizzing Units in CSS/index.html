Sizzing units in css

There are several types of units in css :

  • px - pixels.
  • vw - viewport-width preffered to make sites dynamically active so that scree and color pixels may not burst when going out of monitor resolution length.
  • vh - to adjust height of container as x% of the height of vw
  • em - unit used to adjust font size gives x times of inherited or default value i.e. 16px.
  • rem - gives font size as x% of the root size;
  • vmin - Sometimes width is greater than height but in phone height is greater than width; So in phone - vmin is width, but in computer vmin is height.
  • vmax- almost similar to vmin
  • % - refers to x% to that of parent.
    • and many more complex units are there.....

margin- auto automatically sets the margin both from left and right but it is centered only if it is a block level element ; not works in inline element.

Note : margin auto cannot be applied on span and width can't be applied
